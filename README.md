# Hytale Entity Duplicator
### Fiscally sponsored by HytaleModding Grant Program

The Hytale Entity Duplicator allows modders aims to eliminate modding busy-work by duplicating all assets referenced within a original asset. It does this by taking in a reference asset and duplicating all connected assets required for the full functionality and implementation. That means the asset is ready to go and automatically updated in Hytale so that you can solely focus on the creative aspects of modding like designing, modelling, and programming.

#### How to Install & Run:
1. Look for the Latest on [Releases](https://github.com/Agent40infinity/Hytale-Crop-Duplicator/releases)
2. Unzip and Run "Hytale Crop Duplicator.exe" (Unity Application)
3. Follow the directions labeled on the placeholder texts
   
*(To ensure live updating, ensure the target folder is within "Hytale\UserData\Mods")*

#### How to Extract Hytale Assets:
1. Navigation to "C:\Users\[User]\AppData\Roaming\Hytale\install\release\package\game\latest" or the reletive install location
2. Extract "Assets.zip" to a seperate easy to access location
3. Use this folder for your reference assets when duplicating
   
*(Re-extraction required upon Hytale Updating)*
